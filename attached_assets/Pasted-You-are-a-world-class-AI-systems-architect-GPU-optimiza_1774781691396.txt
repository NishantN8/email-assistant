# ⚡ MODEL ORCHESTRATION ENGINE (CRITICAL)

Implement a decision router:

```ts
function selectModel(task, priority) {
  if (task === "classification") return "local"

  if (task === "summary" && priority < 70)
    return "local"

  if (task === "reasoning" && priority > 70)
    return "cloud"

  if (task === "reply" && priority > 80)
    return "cloud"

  return "local"
}
```
