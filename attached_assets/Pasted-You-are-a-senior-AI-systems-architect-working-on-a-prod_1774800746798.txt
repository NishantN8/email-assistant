# 🧠 1. SPAM HEURISTIC ENGINE

Create:

services/spamHeuristic.js

Logic:

* assign spam_score (0–100)

Rules:

+30 → promotional keywords (sale, offer, discount)
+25 → unsubscribe link present
+20 → bulk email pattern (template similarity)
+15 → suspicious sender/domain
+10 → excessive links/images
+20 → no prior interaction
-25 → user replied before
-30 → trusted sender

If spam_score > 70 → mark as likely spam

---

# 🧠 2. ML SPAM CLASSIFIER

Create:

services/spamClassifier.js

Use lightweight model:

* Naive Bayes / logistic regression (fast inference)

Features:

* subject tokens
* sender domain frequency
* email length
* link count
* historical interaction

Output:

{
spam_probability: 0–1
}

---

# 🧠 3. LLM SPAM JUDGE (GPU-FIRST)

Trigger ONLY if:

* heuristic + ML confidence < threshold

Use Ollama (GPU):

Prompt:

{
"is_spam": true/false,
"category": "promotion | scam | newsletter | irrelevant",
"reason": "...",
"confidence": 0–1
}

Use OpenAI ONLY if:

* CRITICAL ambiguity

---

# 🧠 4. FINAL DECISION ENGINE

Combine:

final_score = weighted(
heuristic_score,
ml_probability,
llm_confidence
)

Decision:

> 85 → DELETE
> 70–85 → JUNK
> 50–70 → BUNDLE (collapse UI)
> <50 → KEEP
